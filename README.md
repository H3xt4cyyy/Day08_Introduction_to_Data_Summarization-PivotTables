# HR Data Analytics & Aggregation

## 📌 Project Overview
This repository contains a data analytics project exploring a 50-employee Human Resources dataset. The objective is to summarize raw data and answer practical business questions using two different analytical approaches: visual aggregation in Excel and backend querying with SQL.

## 📊 Key Insights Explored
* **Departmental Headcount:** Identifying the largest and smallest business units.
* **Payroll Allocation:** Tracking total salary expenditures across different regional offices (Manila, Cebu, Davao).
* **Compensation Analysis:** Calculating average salaries by job position.
* **Hiring Trends:** Visualizing year-over-year recruitment growth from 2016 to 2024.

## 🛠️ Tools & Technologies Used
* **Microsoft Excel:** Advanced Pivot Tables, Cross-Tabs, and Temporal Grouping.
* **SQL (SQLite):** Data aggregation using `GROUP BY`, mathematical functions (`SUM`, `AVG`, `ROUND`), and string manipulation.
